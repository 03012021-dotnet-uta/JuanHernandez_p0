You will enter a start number and a stop number

When a number is a multiple of three, it will print “sweet” instead of the number on the screen.

If the number is a multiple of five, print “salty”(instead of the number) to the screen.

Finally if the number is a multiple of both five and three, print “sweet'N'salty”(instead of the number) to the screen.

The start and stop numbers must be at least 200 apart and no more than 10,000 apart.
